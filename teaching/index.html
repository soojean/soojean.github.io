Teaching

  1. [Fall 2023] EE 488F: Learning Patterns for Autonomous Control. The theme of this course is to identify and use patterns in intelligent autonomous systems. By studying patterns, many control and state-estimation algorithms can be more efficient in data consumption and computation time. Relevant motivating applications include robotic path-planning with AI, fault-tolerant control, and decision-making networks (e.g., vehicle traffic systems, UAV traffic management). This is a topics course based on my own research, and is open to all interested undergraduate, MS, and PhD students.
